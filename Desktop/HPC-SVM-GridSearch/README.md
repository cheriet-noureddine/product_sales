# HPC SVM Grid Search on Adult Income Dataset

This project demonstrates **High Performance Computing (HPC)** with **Support Vector Machines (SVM)** using **Grid Search** on the Adult Income dataset.

## Project Structure

- `serial/` → Standard Python grid search
- `parallel/` → Parallel grid search using joblib
- `data/` → Dataset files
- `results/` → Accuracy + timing outputs

## Setup

```
bash
pip install -r requirements.txt
```

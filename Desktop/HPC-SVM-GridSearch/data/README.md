# Adult Income Dataset

This project uses the **Adult Income dataset** (also known as "Census Income").
It predicts whether a person earns more than $50K/year based on census features.

## Instructions

1. Download the dataset from UCI ML Repository:
   https://archive.ics.uci.edu/ml/datasets/adult
2. Save it as `adult.csv` inside the `data/` folder.
3. Ensure the file path matches `data/adult.csv`.
